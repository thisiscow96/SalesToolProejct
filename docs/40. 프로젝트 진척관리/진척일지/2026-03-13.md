## 배포 · 도메인 · 이메일

- **백엔드 배포**
  - Render Web Service + Render Postgres (Railway에서 이전)
  - Start Command: `npm run migrate && npm start` (마이그레이션 자동 적용)
- **프론트 배포**
  - Vercel (`sales-tool-proejct.vercel.app`), `vercel.json` rewrite로 Render API 연동
- **도메인 (garaksalestool.com)**
  - 가비아에서 도메인 등록 완료
  - **가비아 DNS 4개 등록**: Vercel용 A `@` → `216.198.79.1` / Resend용 MX `send`, TXT `send`(SPF), TXT `resend._domainkey`(DKIM)
  - Vercel에 커스텀 도메인 추가, DNS 전파 후 **Valid Configuration** 완료 → **https://garaksalestool.com** 접속 가능
- **이메일 (Resend)**
  - Resend 연동: `RESEND_API_KEY` 설정 시 Resend API로 인증 메일 발송, 미설정 시 로컬 Gmail SMTP
  - Resend Domains에 **garaksalestool.com** 추가, MX/SPF/DKIM 등록 후 **도메인 인증(Verify) 완료**
  - Render Environment `RESEND_FROM` = `noreply@garaksalestool.com` 설정 후 Redeploy → 아무 이메일로 인증 메일 발송 가능
- **문서**
  - `docs/아키텍쳐 구성/도메인/Resend-설정-가이드.md`, `docs/아키텍쳐 구성/도메인/Resend-도메인인증-salestool.md`, `docs/아키텍쳐 구성/도메인/garaksalestool.com-도메인-연결.md`
