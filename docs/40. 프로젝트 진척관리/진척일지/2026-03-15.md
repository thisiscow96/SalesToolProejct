- **배포 환경**
  - Railway 프로젝트에 GitHub 레포(`SalesToolProejct`) 연결
  - 백엔드 서비스: Root `server`, Start `npm start`, Variables에 `DATABASE_URL` (Postgres)
  - PostgreSQL(Postgres-1sVd) 사용, **Generate Domain** 으로 백엔드 공개 URL 확보
- **코드**
  - `app.listen(port, '0.0.0.0', ...)` 적용, `DATABASE_URL`이 postgres/postgresql로 시작할 때만 DB 사용, 프로덕션 Postgres SSL·Pool error 핸들링
