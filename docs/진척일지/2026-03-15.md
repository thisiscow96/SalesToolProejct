## Railway 백엔드 배포 완료

- **배포 환경**
  - Railway 프로젝트에 GitHub 레포(`SalesToolProejct`) 연결
  - 백엔드 서비스: Root `server`, Start `npm start`, Variables에 `DATABASE_URL` (Postgres 연결)
  - PostgreSQL: Postgres-1sVd 서비스 사용, 내부 URL `postgres-1svd.railway.internal:5432/railway`
  - **Generate Domain** 으로 백엔드 공개 URL 확보 (예: switchyard.proxy.rlwy.net)

- **코드 수정**
  - 서버가 Railway 지정 포트에서 외부 접속을 받도록 `app.listen(port, '0.0.0.0', ...)` 적용
  - `DATABASE_URL` 이 `postgres://` 또는 `postgresql://` 로 시작할 때만 DB 연결에 사용 (앱 URL 혼동 방지)
  - 프로덕션에서 Postgres SSL 옵션 적용, Pool `error` 이벤트 핸들링으로 비정상 종료 방지

- **확인**
  - 백엔드 URL 접속 시 `{"ok":true,"message":"Sales tool backend is running"}` 응답 확인됨
