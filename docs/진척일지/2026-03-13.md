## 현재까지 완료된 작업 (도메인·이메일)

- **도메인 (garaksalestool.com)**
  - 가비아에서 도메인 등록 완료
  - **가비아 DNS 4개 등록 완료**
    - **Vercel용 1개**: A `@` → `216.198.79.1` (사이트 연결)
    - **Resend용 3개**: MX `send` → `feedback-smtp.ap-northeast-1.amazonses.com`(우선순위 10), TXT `send`(SPF), TXT `resend._domainkey`(DKIM)
  - Vercel에 커스텀 도메인 `garaksalestool.com` 추가, DNS 전파 후 **Valid Configuration** 인증 완료
  - 접속: **https://garaksalestool.com** 으로 판매툴 접속 가능
- **Resend**
  - Resend Domains에 **garaksalestool.com** 추가, 가비아에 MX/SPF/DKIM 3개 등록 후 **도메인 인증(Verify) 완료**
  - Render **Environment** 에 **RESEND_FROM** = `noreply@garaksalestool.com` 설정 후 Redeploy → 아무 이메일로 인증 메일 발송 가능
- **환경 변수**
  - `docs/환경 변수설정/README.md` — garaksalestool.com 기준 VITE_API_URL·RESEND_FROM·가비아 DNS 4개 정리 반영
