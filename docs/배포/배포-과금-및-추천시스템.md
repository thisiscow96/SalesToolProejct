# 배포 과금 및 추천 시스템 요약

※ 가격·한도는 서비스 정책에 따라 바뀔 수 있음. 최신은 각 사이트에서 확인.

---

## 1. 지금 쓰는 시스템 (과금)

### Render (백엔드 + DB)
| 구분 | 내용 |
|------|------|
| **무료** | Web Service 750시간/월, 15분 미사용 시 슬립 → 콜드스타트 30초~1분. Postgres 무료는 **90일 후 만료** 가능. |
| **유료** | Starter $7/월~, Standard $25/월~ (RAM·CPU 상향). |
| **참고** | [Render Pricing](https://render.com/pricing) |

### Vercel (프론트)
| 구분 | 내용 |
|------|------|
| **무료 (Hobby)** | 개인/비상업용. 100만 함수 호출/월, 100GB 대역 등. **상업용은 Pro 이상 필요.** |
| **유료** | Pro $20/사용자/월. |
| **참고** | [Vercel Pricing](https://vercel.com/pricing) |

## 2. 이메일 발송 (추천)

### Resend (추천 — Render/Vercel에서 SMTP 대신 사용)
| 구분 | 내용 |
|------|------|
| **무료** | **일 100통, 월 3,000통.** 도메인 1개, 팀 1명. |
| **유료** | Pro 등 월 구독 또는 사용량 과금 ($0.90/1,000통 등). |
| **역할** | Render/Railway는 SMTP(587/465) 막힘 → Gmail 직접 연결 불가. Resend는 **HTTPS API**라서 PaaS에서 인증 메일 발송 가능. |
| **참고** | [Resend Pricing](https://resend.com/pricing) |

### Railway
| 구분 | 내용 |

## 4. Gmail SMTP를 꼭 쓸 때 (대안)

### VPS (예: DigitalOcean)
| 구분 | 내용 |
|------|------|
| **가격** | Basic Droplet **$4~6/월** (512MB~1GB RAM) 등. |
| **역할** | SMTP 포트 막지 않음 → **Gmail SMTP 그대로 사용 가능.** 대신 서버(OS·Node·DB) 직접 관리 필요. |
| **참고** | [DigitalOcean Droplet Pricing](https://www.digitalocean.com/pricing/droplets) |

## 5. 추천 조합 요약

| 목적 | 추천 | 예상 월 비용 (무료 범위) |
|------|------|---------------------------|
| **지금처럼 Render + Vercel 유지, 이메일만 되게** | **Resend** 추가 | $0 (무료 한도 내: 일 100통) |
| **Gmail SMTP만 고수** | 백엔드를 **VPS**(DigitalOcean 등)로 이전 | 약 $4~6/월 (Droplet 최저) |
| **전부 무료로 유지** | Render 무료 + Vercel Hobby + Resend 무료 (상업용 아니면) | $0 (한도·제한 내) |

인증 메일만 보낸다면 **Resend 무료**로 충분한 경우가 많고, Gmail SMTP를 반드시 써야 하면 **VPS**로 백엔드만 옮기는 선택이 필요함.
