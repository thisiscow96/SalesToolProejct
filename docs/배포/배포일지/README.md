## 작성 규칙

- **배포ID**: `DEP-001` 부터 순번 (새 배포 시 다음 번호 부여)
- **날짜**: 배포 수행일 (YYYY-MM-DD)
- **시스템**: 배포를 수행한 주체 (GitHub, Vercel, Render, 가비아(DNS), Resend 등)
- **배포 및 수정내용**: 변경·배포한 내용을 구체적으로 기록
- **반영된 시스템**: 이 배포가 **어디까지 올라갔는지** (예: GitHub만 / GitHub → Vercel / GitHub → Vercel, Render). Git push 시 연동된 서비스 자동 배포 여부를 대시보드에서 확인 후 기입.
