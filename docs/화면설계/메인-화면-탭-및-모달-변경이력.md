# 메인 화면 · 탭 · 모달 변경이력

**대상:** 로그인 후 메인 화면 (`Main.jsx`) — 상단 탭 + 탭별 필터·테이블·모달.

---

## 1. 공통

| 요소 | 역할 |
|------|------|
| 상단 탭 | `재고현황` · `매입정보` · `매출정보` · `수금정보` · `폐기정보` · (관리자만) `상품 마스터` |
| 헤더 | 로고·로그아웃 |
| `main-content` | 선택 탭 본문 |

---

## 2. 탭별 요약 (최신 UI 기준)

### 재고현황

- 상품명, 매입 거래처, 수량, 단위, 매입일(일자만) 표시.
- **변경 없음** (최근 대규모 변경 없음).

### 매입정보

- 기간·제품 필터, **검색**, **매입 등록**(모달), **선택 매출 전환**(모달).
- 매입 등록 모달: 행 추가·복제·매입처·상품·수량·단가·매입일.

### 매출정보

- 기간·거래처·**미수·일부 포함** 체크, **검색**, **수금 등록**, **환불 처리**.
- **수금 등록** 모달: 거래처·수금일시·미수 매출 테이블 — 매출일은 **줄바꿈·전체 표시**(가로 스크롤 제거), **이번 수금**은 **숫자만·최대 9자리**.
- **환불 처리** 모달: **거래처 검색**, **사유 통일** 체크, 수금 발생(`paid_amount>0`)·잔여 수량 있는 매출만 — 테이블 가로 스크롤·열 최소폭, 매출일은 **일시만**(매출수량 보조문구 제거), **사유**는 행 선택과 무관 입력 가능, 환불 요청수량 **최소 3자리**, 환불일 UI 없음(저장 시 당일).

### 수금정보

- 기간·거래처·검색. 수금/환불 구분·금액·비고 목록.

### 폐기정보

- 목록: 기간·검색, **폐기 등록** 버튼.
- **폐기 등록** 모달 (**변경됨**):
  - **이전:** 행마다 상품 셀렉트 + 행 추가·복제·삭제.
  - **현재:** 모달 오픈 시 **재고** 로드 → 상단 **판매처(매입처) 검색** + **매입일** — 값이 있는 조건만 AND 필터, 둘 다 비우면 전체 재고.
  - 테이블: 체크·상품·재고·판매처·매입일·폐기 수량·폐기일·사유(상품 변질 / 상품 폐기 / 직접 입력)·비고.
  - **복제 버튼 제거.**

### 상품 마스터 (관리자)

- 카테고리·이름 검색, 목록, **새로 만들기** 모달(단건/다건).
