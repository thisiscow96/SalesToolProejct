# Overview

본 프로젝트는 실제 판매 관리 업무에서 발생하는 반복 입력과 데이터 분산 문제를 해결하기 위해 시작되었습니다.

기존 시스템은 매입, 매출, 수금, 폐기 데이터가 각각 분리되어 관리되고 있었고, 동일한 정보를 여러 번 입력해야 하는 구조로 인해 업무 피로도가 높고 운영 효율이 떨어지는 문제가 있었습니다.

이를 개선하기 위해 단순 기능 추가가 아닌, **업무 흐름 자체를 하나의 체계로 통합하고 운영 가능한 구조로 재설계**하는 것을 목표로 시스템을 구축했습니다.

결과적으로, 데이터 흐름을 정리하고 사용자 중심의 입력 구조를 반영하여 실제 운영 환경에서 활용 가능한 판매 관리 시스템을 구현했습니다.
