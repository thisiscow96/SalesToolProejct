# Implementation

## Architecture

- Backend: Node.js (Express)
- Frontend: React (Vite)
- Database: PostgreSQL

## 주요 구현 내용

### 1. 통합 관리 UI 구성

- 매입/매출/수금/폐기 데이터를 하나의 흐름에서 관리
- 모달 기반 입력 구조로 사용자 편의성 개선
- 입력값 가시성 확보 및 반응형 UI 적용

### 2. 수금 관리 기능 개선

- 미수금 초과 입력 방지 로직 적용
- 날짜 입력 및 컬럼 구조 개선

### 3. 폐기 처리 프로세스 개선

- 기간 기반 조회 기능 (From ~ To)
- 처리 대상 선택 및 검증 로직 정리

### 4. 운영 및 배포 체계 구축

- GitHub 기반 형상관리
- Render / Vercel을 활용한 배포 환경 구성
- 배포 이력 및 변경 사항 기록 관리

### 5. 개발 프로세스 개선 (Agent 활용)

- 분석 → 설계 → 구현 → 검증 → 문서화 반복 구조
- 규칙 기반 개발 환경 구성으로 품질 기준 유지
