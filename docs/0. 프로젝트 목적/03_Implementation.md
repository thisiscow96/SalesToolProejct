# Implementation

## Architecture

- Backend: Node.js (Express) — 레포: [`server/`](../../server/)
- Frontend: React (Vite) — 레포: [`web/`](../../web/)
- DB: PostgreSQL

## 주요 구현

- 매입/매출/수금/폐기 통합 관리 UI
- 수금 입력 검증 로직
- 폐기 처리 프로세스 개선
- 운영 문서 및 배포 체계 구축

상세 DDL·API·화면 설계는 [`docs/10. 포로젝트 분석설계/`](../10.%20포로젝트%20분석설계/) 및 [`docs/99. 배포관리/`](../99.%20배포관리/) 를 참고하세요.
