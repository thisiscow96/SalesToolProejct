## 삽화(SVG)

**`images/*.svg`** — 매뉴얼용 **자리표시** 이미지(텍스트 안내). Git에 포함되어 있으며 GitHub에서 Markdown 미리보기 시 표시됩니다.

## 실제 스크린샷(PNG)으로 교체할 때

1. 브라우저에서 캡처 후 `images/` 에 저장.
2. `운영자-사용자-매뉴얼.md` 에서 해당 줄의 `.svg` 를 `.png` 로 바꾸거나, 같은 베이스 이름의 PNG를 쓰도록 링크 수정.

(선택) Windows에서 래스터 PNG를 만들려면 `scripts/Generate-PlaceholderImages.ps1` 는 **UTF-8 BOM** 저장 후 실행해야 한글이 깨지지 않을 수 있습니다. **SVG가 기본**이므로 스크립트는 필수 아님.
