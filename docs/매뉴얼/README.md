## 스크린샷

실제 화면 캡처 PNG는 **`images/`** 폴더에 저장합니다.

1. 브라우저에서 해당 화면을 연 뒤 스크린샷 저장.
2. 파일명은 매뉴얼 본문의 `![설명](images/파일명.png)` 와 맞출 것.
3. Git에 커밋 시 `images/` 가 함께 올라가도록 추가.

`images/` 에 파일이 없어도 Markdown 미리보기에서 깨질 수 있으니, 배포 전에 캡처를 채워 넣는 것을 권장합니다.
