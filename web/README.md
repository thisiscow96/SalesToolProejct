# web (React)

판매툴의 **PC 웹 프론트엔드**를 담당하는 디렉터리입니다.

## 앞으로의 계획

- React 기반으로 화면을 구성합니다.
- 빌드 도구는 Vite 또는 Next.js 중 하나를 선택해 초기 세팅할 수 있습니다.

## 초기 세팅 예시 (Vite + React)

로컬 터미널에서 다음 명령으로 Vite 템플릿을 생성할 수 있습니다.

```bash
cd "C:\Users\gunme\Desktop\건우\프로젝트\3. 판매툴"
npm create vite@latest web -- --template react
```

생성 후에는 `web/` 디렉터리에서 의존성을 설치하고 개발 서버를 실행합니다.

```bash
cd web
npm install
npm run dev
```
