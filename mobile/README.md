# mobile (Expo)

판매툴의 **모바일/멀티플랫폼 앱**을 담당하는 디렉터리입니다.

## 앞으로의 계획

- Expo(React Native)를 사용해 iOS/Android/웹(선택)을 지원합니다.

## 초기 세팅 예시 (Expo)

로컬 터미널에서 다음 명령으로 Expo 앱을 생성할 수 있습니다.

```bash
cd "C:\Users\gunme\Desktop\건우\프로젝트\3. 판매툴"
npx create-expo-app mobile
```

생성 후에는 `mobile/` 디렉터리에서 앱을 실행합니다.

```bash
cd mobile
npm install   # 또는 yarn
npx expo start
```
