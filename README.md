## 판매툴 프로젝트

PC 웹과 모바일 앱(Expo)을 함께 개발하는 판매 관리 툴입니다.

### 기술 스택 (계획)
- 백엔드: Node.js (예정) + PostgreSQL
- 프론트엔드(웹): React
- 모바일/멀티플랫폼: Expo (React Native)

### 개발 환경
- Git: 버전 관리
- GitHub: 원격 저장소 및 협업

### 문서 · 이력
- **문서 목차**: [docs/README.md](docs/README.md) — 번호 폴더(배포·매뉴얼·테스트 등) 안내.
- **연동 이력(시행착오)**: [docs/00. 프로젝트 관리/연동-이력.md](docs/00.%20프로젝트%20관리/연동-이력.md) — Git, Node, 백엔드, DB 연동 시 겪은 문제와 해결 방법 정리.
- **Object 설계서**: [docs/10. 포로젝트 분석설계/object 설계서/](docs/10.%20포로젝트%20분석설계/object%20설계서/) — DB·업무 객체 설계. DDL은 [server/schema.sql](server/schema.sql).
- **진척일지**: [docs/40. 프로젝트 진척관리/진척일지/](docs/40.%20프로젝트%20진척관리/진척일지/) — 날짜별 작업 내역 간략 기록.

### 기본 사용 방법
1. 이 저장소를 클론합니다.
2. Node.js와 패키지 매니저(npm 또는 yarn)를 설치합니다.
3. 이후 백엔드/프론트엔드/모바일 프로젝트 구조를 추가로 정의할 예정입니다.
