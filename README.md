## 판매툴 프로젝트

PC 웹과 모바일 앱(Expo)을 함께 개발하는 판매 관리 툴입니다.

### 기술 스택 (계획)
- 백엔드: Node.js (예정) + PostgreSQL
- 프론트엔드(웹): React
- 모바일/멀티플랫폼: Expo (React Native)

### 개발 환경
- Git: 버전 관리
- GitHub: 원격 저장소 및 협업

### 문서 · 이력
- **연동 이력(시행착오)**: [docs/연동-이력.md](docs/연동-이력.md) — Git, Node, 백엔드, DB 연동 시 겪은 문제와 해결 방법 정리. GitHub에 커밋하여 이력 관리.
- **Object 설계서**: [docs/object 설계서/](docs/object%20설계서/) — 현재까지의 DB·업무 객체 설계 (엔티티, 흐름, 화면 고려). DDL은 [server/schema.sql](server/schema.sql).
- **진척일지**: [docs/진척일지/](docs/진척일지/) — 날짜별 작업 내역(연동, DB 설계 등) 간략 기록.

### 기본 사용 방법
1. 이 저장소를 클론합니다.
2. Node.js와 패키지 매니저(npm 또는 yarn)를 설치합니다.
3. 이후 백엔드/프론트엔드/모바일 프로젝트 구조를 추가로 정의할 예정입니다.
